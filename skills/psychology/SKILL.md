# 生命阶段锚点图谱（引擎 2 核心数据）

根据用户年龄定位阶段，用该阶段的断言制造"读心"效果。
**所有断言必须是陈述句，不是疑问句。**

## 阶段速查表

| 年龄段 | 阶段 | 核心关切 | 高命中断言（选 1-2 条用） |
|-------|------|---------|----------------------|
| 15-18 | 高中生 | 学业、高考 | "你心里有一个人，但你告诉自己现在不是时候" / "你有一门课一直不稳定，这让你心里没底" / "你父母对你期望很高，你嘴上说不在乎，但心里压力很大" |
| 18-22 | 大学生 | 方向感、恋爱 | "你现在学的专业，可能不是你最初想选的" / "你大一大二比较迷茫，到大三才慢慢清醒" / "你身边有人已经拿到了你想要的东西，你替他高兴，但也有一种说不出的滋味" |
| 22-25 | 初入社会 | 工作方向、收入 | "你第一份工作，干了不到两年就开始动摇了" / "你现在赚的钱，和你觉得自己值的钱，中间有一个落差" / "你父母希望你回老家或者考公，但你不甘心" |
| 25-28 | 第一个十字路口 | 婚恋压力、事业确认 | "你谈过一段认真的感情，但没走到最后，那个人偶尔还会出现在你脑海里" / "你对现在的工作不是讨厌，而是觉得'就这样了吗'" / "你有一个想法一直没行动——可能是创业、转行、或去另一个城市" |
| 28-32 | 关键抉择期 | 结婚/生育/买房 | "你不是不想结婚，而是不想将就" / "你最近在算一笔账——这个城市值不值得留下来" / 已婚："你和另一半最近因为一件说大不大的事有过争执" |
| 32-36 | 角色转换期 | 职业天花板、健康信号 | "你最近体检有一两项指标不太好，你嘴上说没事，但心里记着" / "你最近开始关注养生、健身了，以前从来不在意这些" / "你在某个深夜想过'如果重新来一次我会怎么选'" |
| 36-40 | 中年初现 | 孩子教育、婚姻质量 | "你和另一半已经很久没有认真谈过心了" / "你身边有一个同龄人最近做了一件你暗暗羡慕的事" / "你偶尔会有一种时间在加速的感觉" |
| 40-45 | 中年深水区 | 财务安全、孩子青春期 | "你的孩子最近让你头疼——不是学习的事，而是你说什么他都不听" / "你父母最近有一次住院或体检异常，那之后你心里多了一根弦" / "你开始对'安全感'有了不一样的理解" |
| 45-50 | 知天命前夜 | 退休规划、子女前途 | "你最近有一种'被时代抛弃'的感觉——不是你不行了，而是游戏规则变了" / "你有一件年轻时没做成的事，最近又开始想了" |
| 50-55 | 退休过渡 | 健康、子女婚嫁 | "你退下来之后有一段时间不知道干什么好" / "你的孩子有自己的生活了，你想靠近又怕打扰" |
| 55-65 | 银发期 | 家庭和谐、健康 | "你有一个老朋友最近生了病或者走了，这件事对你触动很大" / "你对很多事情看开了，但'看开'和'释怀'之间还有一段距离" |
| 65+ | 晚年 | 健康、身后事 | "你最怕的不是自己生病，而是生病之后给孩子们添负担" / "你有一些话想对孩子说，但一直没找到合适的时机" |

## 性别修正

- 男性通用："你命局中透出一种'孤君'的象，外表看着能扛事，但很多情绪都在自我内耗"
- 女性通用："你一直在为别人付出——家人、伴侣、朋友——但你觉得他们未必都领情"

## 跨阶段联动（反推亲属）

用户的年龄暗含父母和子女的阶段，可制造跨域命中：

| 用户年龄 | 父母约 | 子女约 | 可断言 |
|---------|-------|-------|-------|
| 25-30 | 50-60 | 无/幼儿 | 父母退休适应、催婚 |
| 30-35 | 55-65 | 0-8 | 父母健康转折 |
| 35-40 | 60-70 | 5-12 | 父母重病风险、子女教育 |
| 40-45 | 65-75 | 10-18 | 父母养老、子女青春期 |
| 45-50 | 70-80 | 15-22 | 父母高龄、子女高考/就业 |

## 通用兜底断言（任何年龄都适用）

- "你对人真诚，但经常感觉付出和回报不成正比"
- "你在公众场合很从容，但独处时会有一些别人不知道的焦虑"
- "你有过一次很重要的信任被辜负的经历，从那以后你在人际关系上更加谨慎了"
- "你内心深处其实是一个很柔软的人，只是不太愿意在别人面前展示脆弱"
