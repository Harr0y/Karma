# Agent Reach - 互联网能力

为命理咨询提供互联网信息获取能力，包括全网语义搜索和多平台链接读取。

## 核心功能

| 功能 | 工具 | 使用场景 |
|------|------|---------|
| 🔍 全网搜索 | Exa MCP | 搜索年份大事、行业趋势、地域特点 |
| 🌐 网页读取 | Jina Reader | 读取用户分享的网页链接 |
| 📺 视频信息 | yt-dlp | 提取 YouTube/B站视频信息和字幕 |
| 📦 GitHub | gh CLI | 读取 GitHub 仓库和 Issue |

## 何时使用

### 全网搜索 (Exa MCP)

**使用场景：**
- 用户提到某年某事，需要验证背景 → 搜索 "1991年 中国 经济大事"
- 了解地域特点 → 搜索 "湖南 人文特点"
- 行业趋势分析 → 搜索 "2024年 教培行业 政策"
- 历史大事件 → 搜索 "2008年 汶川地震 影响"

**调用方式：**
```bash
mcporter call exa.web_search_exa query="搜索内容" numResults=3
```

**示例：**
```
用户: 我是91年出生的，湖南人

搜索: 1991年 中国 经济发展 湖南
结果: 1991年浦东开发开放、深圳证券交易所开业、湖南经济发展...
融入断言: "91年属羊，那一年浦东刚开发开放，整个社会机会多..."
```

### 网页读取 (Jina Reader)

**使用场景：**
- 用户分享网页链接 → 读取正文内容
- 需要了解某个网页的内容

**调用方式：**
```bash
curl -s "https://r.jina.ai/URL"
```

**示例：**
```
用户分享: https://example.com/article

读取: curl -s "https://r.jina.ai/https://example.com/article"
结果: 网页正文内容...
```

### 视频信息 (yt-dlp)

**使用场景：**
- 用户分享 YouTube/B站视频 → 提取视频信息和字幕

**调用方式：**
```bash
yt-dlp --dump-json "视频URL"
```

**返回信息：**
- 视频标题、描述
- 时长、观看数
- 字幕（如有）

### GitHub (gh CLI)

**使用场景：**
- 读取公开仓库信息
- 查看 Issue 内容

**调用方式：**
```bash
gh repo view owner/repo
gh issue view 123 --repo owner/repo
```

## 使用原则

### 1. 搜索是辅助，不是替代

- 搜索结果用于**验证和补充**断言
- 核心仍是八字推理和冷读技巧
- 不要过度依赖搜索

### 2. 自然融入，不暴露搜索过程

**❌ 错误：**
```
"我搜索了一下，1991年发生了..."
```

**✅ 正确：**
```
"91年那会儿，浦东刚开发开放，整个社会机会多。你出生在那样一个年份..."
```

### 3. 控制搜索频率

- 每轮对话 1-2 次搜索足够
- 不要每句话都搜索
- 优先使用已有信息

## 工具调用详情

### Exa 全网搜索

```bash
mcporter call exa.web_search_exa \
  query="搜索内容" \
  numResults=3 \
  type="auto"
```

**参数说明：**
- `query`: 搜索内容（必需）
- `numResults`: 返回结果数（默认 8，建议 3-5）
- `type`: 搜索类型（auto/fast）

### Jina 网页读取

```bash
curl -s "https://r.jina.ai/URL"
```

- 支持任意网页 URL
- 返回 Markdown 格式正文
- 免费，无需 API Key

### yt-dlp 视频提取

```bash
yt-dlp --dump-json "URL"
```

- 支持 YouTube、B站等 1800+ 平台
- 返回 JSON 格式视频信息
- 可提取字幕（如有）

## 使用场景示例

### 场景 1: 搜索背景信息

**用户：** 我是91年出生的，湖南人

**Agent 内心独白：**
```
需要了解1991年的背景，融入断言增强可信度。
```


### 场景 2: 读取用户分享的链接

**用户：** 帮我看看这个小红书笔记说得对不对 [链接]

**Agent 内心独白：**
```
用户分享了一个链接，需要读取内容。
```

**调用：**
```bash
curl -s "https://r.jina.ai/小红书链接"
```

**回复：**
```
看了这篇笔记，它说的是...（根据内容回应）
```

### 场景 3: 了解行业背景

**用户：** 我是做教培的

**Agent 内心独白：**
```
教培行业在2021年有重大政策变化，需要了解背景。
```

**调用：**
```bash
mcporter call exa.web_search_exa query="2021年 教培行业 双减政策 影响" numResults=3
```

**融入回复：**
```
教培这行，21年那会儿是个坎。"双减"政策出来后，整个行业都变了。
你这个盘，那几年应该感觉到了压力...
```
