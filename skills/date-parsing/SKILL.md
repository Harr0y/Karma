# 生辰解析指南

客户给出的生辰信息可能以各种形式表述。你的任务是将其解析为标准格式，供八字排盘使用。

## 公历表述

常见格式：
- `1990年5月15日` / `1990-05-15`
- `90年农历八月十七` → 注意这里说的是农历
- `5月15号` / `五月十五`

## 农历表述

关键词识别：
- `农历` / `阴历` / `旧历` → 农历
- `腊月` = 农历十二月
- `正月` = 农历一月
- `冬月` = 农历十一月

常见格式：
- `农历八月十七` / `阴历8月17`
- `腊月初八` / `正月十五`
- `甲子年八月十七` → 年份用干支纪年

## 时辰表述

中国传统时辰与小时对应：

| 时辰 | 时间段 | 别称 |
|------|--------|------|
| 子时 | 23:00-01:00 | 半夜、子夜 |
| 丑时 | 01:00-03:00 | 鸡鸣 |
| 寅时 | 03:00-05:00 | 黎明 |
| 卯时 | 05:00-07:00 | 日出、破晓 |
| 辰时 | 07:00-09:00 | 食时、早食 |
| 巳时 | 09:00-11:00 | 隅中 |
| 午时 | 11:00-13:00 | 日中、正午 |
| 未时 | 13:00-15:00 | 日昳 |
| 申时 | 15:00-17:00 | 晡时 |
| 酉时 | 17:00-19:00 | 日入、黄昏 |
| 戌时 | 19:00-21:00 | 黄昏 |
| 亥时 | 21:00-23:00 | 人定 |

常见表述：
- `中午12点` / `午时`
- `早上6点` / `卯时`（日出前后）
- `傍晚` / `酉时`
- `半夜` / `子时`
- `下午3点` / `申时`
- `凌晨2点` / `丑时`

## 年份表述

### 生肖年
- `属马的` / `属马` → 需要推断是哪一年
- 常见生肖年：鼠(2020)、牛(2021)、虎(2022)、兔(2023)、龙(2024)、蛇(2025)...

### 干支年（甲子纪年）
- `甲子年` → 60年一个周期，需结合上下文推断
- `庚子年` → 2020年是庚子年
- `壬寅年` → 2022年是壬寅年

### 年份推断
如果客户说年龄而非年份：
- `我今年35岁` → 当前年份 - 35（或36，需考虑虚岁）
- 中国传统有"虚岁"概念，可能比实际大1岁

## 模糊时间

- `端午前后` → 农历五月初五前后
- `中秋` → 农历八月十五
- `过年那会儿` → 农历正月初一前后
- `立春` → 通常2月4日左右
- `清明` → 通常4月5日左右

## 解析策略

1. 优先获取精确信息：公历日期 + 具体时间
2. 如果只有农历，需要转换为公历（可用工具）
3. 时辰模糊时，取中间值（如"中午"取12点）
4. 记录不确定性，后续对话中可以确认

## 调用八字排盘

当解析完成后，调用 `bazi_calculator` 工具：

```
调用工具: bazi_calculator
参数: {
  "birthDate": "1990-05-15T06:00:00",
  "gender": "male"
}
```

或者中文格式：

```
调用工具: bazi_calculator
参数: {
  "birthDate": "1990年5月15日早上6点",
  "gender": "female"
}
```
