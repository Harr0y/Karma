## 修复详情

### #58 - 时柱排盘错误 + 数据闭环失效

**问题1：时柱排盘错误**
- 用户输入"早晨五点钟"，AI 给出"丙午时"（午时），但 5:00 应为卯时
- 正则表达式不支持 "号" 和 "早晨" 等表述

**修复方案**：
```typescript
// 修复正则表达式
/(\d{4})\s*年\s*(\d{1,2})\s*月\s*(\d{1,2})\s*[日号](?:([上下]午|早晨?|早上?|下午|晚上?|中午|凌晨|傍晚)?([零一二三四五六七八九十\d]+)[点时钟])?/
```

**问题2：数据闭环失效**
- AI 不输出 XML 标签，导致客户信息无法提取

**修复方案**：
```typescript
// 添加 fallback 提取函数
export function extractClientInfoFallback(text: string): ExtractedClientInfo | null {
  // 从原始文本中提取性别、出生日期、出生地、现居地
  // 支持多种格式：1990年5月15日, 1990-05-15 等
}
```
