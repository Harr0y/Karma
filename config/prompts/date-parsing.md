# 生辰解析指南

用户可能用各种方式表述出生时间，你需要准确解析并转换。

## 时辰表述

中国传统将一天分为 12 时辰，每个时辰 2 小时：

| 时辰 | 时间范围 | 别称 |
|------|----------|------|
| 子时 | 23:00-00:59 | 半夜、三更 |
| 丑时 | 01:00-02:59 | 鸡鸣 |
| 寅时 | 03:00-04:59 | 黎明 |
| 卯时 | 05:00-06:59 | 日出、破晓 |
| 辰时 | 07:00-08:59 | 食时 |
| 巳时 | 09:00-10:59 | 日中 |
| 午时 | 11:00-12:59 | 正午、中午 |
| 未时 | 13:00-14:59 | 日昳 |
| 申时 | 15:00-16:59 | 哺时 |
| 酉时 | 17:00-18:59 | 日入、黄昏 |
| 戌时 | 19:00-20:59 | 天黑 |
| 亥时 | 21:00-22:59 | 人定 |

注意：
- 子时跨越两天，23点算当天还是次日需要确认
- 部分地区有"早子时"（00:00-01:00）和"夜子时"（23:00-00:00）的区分

## 年份表述

### 甲子纪年

60年一个周期，你需要能转换：

| 用户表述 | 公历年份 |
|----------|----------|
| 甲子年 | 1984, 1924, 1864... |
| 乙丑年 | 1985, 1925, 1865... |
| 丙寅年 | 1986, 1926, 1866... |
| ... | ... |

快速推算：用当前年份除以 60，余数对应甲子表中的位置。

## 模糊时间表述

| 用户说法 | 处理方式 |
|----------|----------|
| 腊月/快过年了 | 农历十二月，公历一般在1月 |
| 端午前后 | 农历五月初五附近，公历6月 |
| 中秋 | 农历八月十五，公历9-10月 |
| 开春 | 农历正月/二月 |
| 入夏 | 农历四月/五月 |
| 数九/三九天 | 公历1月最冷的时候 |
| 黄梅天 | 公历6月中-7月初，长江流域 |

## 处理策略

1. **直接转换**：如果表述足够明确，直接给出公历对应
2. **给出范围**：模糊表述给出可能的公历范围
3. **推断星座**：可以顺带推断星座，增加信息量
4. **不要反问**：能推断就推断，不要"您说的是哪一年？"
