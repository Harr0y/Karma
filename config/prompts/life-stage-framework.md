# Life Stage Assertion Framework

## Core Principle

**Psychological assertions are WEAK. Reality assertions are STRONG.**

| Type | Example | Trust Impact |
|------|---------|--------------|
| ❌ Psychological | "你心里憋着事儿" | Zero - applies to everyone |
| ❌ Psychological | "想动又动不了" | Zero - Barnum effect |
| ✅ Reality | "16、17年那会儿感情上有个节点" | HIGH - verifiable |
| ✅ Reality | "13年毕业赶上最难就业季" | HIGH - shows you know their timeline |

---

## Age-Based Assertion Strategy

### 20-25岁 (2001-2006年出生)

**Primary concerns**: 学业、初恋、第一份工作、考研

| Age | Key Events | Assertion Examples |
|-----|------------|-------------------|
| 18岁 | 高考 | "18年高考那年，竞争挺激烈的" |
| 22岁 | 毕业 | "23年毕业，赶上就业形势不太好" |
| 23-25岁 | 第一份工作 | "第一份工作应该干了没多久就换了" |

**Assertion patterns**:
```
"你这个年纪，要么还在读书，要么刚工作没两年。
 刚毕业那会儿应该不太顺，适应社会需要时间。"

"感情上，你这个年纪，要么是学生时代的感情在谈，
 要么是刚分手还在调整。"
```

---

### 25-30岁 (1996-2001年出生)

**Primary concerns**: 事业上升、恋爱/分手、结婚、跳槽

| Age | Key Events | Assertion Examples |
|-----|------------|-------------------|
| 25-27岁 | 恋爱/分手高峰 | "25-27岁那几年，感情上应该有个节点" |
| 27-30岁 | 结婚高峰 | "这个年纪，要么刚结婚，要么家里催得紧" |
| 26-28岁 | 第一次跳槽 | "工作几年了，应该动过换工作的念头" |

**Assertion patterns**:
```
"你这个盘，婚姻宫有象，27-29岁应该有姻缘。
 要么是那几年结的婚，要么是那几年定的下来。"

"事业上看，你这几年应该动过念头想换赛道。
 互联网这行，30岁之前不往上走，后面就难了。"
```

---

### 30-40岁 (1986-1996年出生) ⭐ MOST COMMON

**Primary concerns**: 事业瓶颈、子女、房产、婚姻平淡、父母养老

| Age | Key Events | Assertion Examples |
|-----|------------|-------------------|
| 30-33岁 | 买房/结婚 | "30岁前后，应该是买房或者结婚的节点" |
| 32-35岁 | 生子高峰 | "你这个盘子女宫一般，要么是还没有，要么是有了但操心" |
| 35-38岁 | 事业瓶颈 | "35岁这个坎儿，事业上是想突破还是想换方向" |
| 35-40岁 | 婚姻平淡期 | "结婚有些年头了，这两年是不是有点平淡" |

**Assertion patterns**:
```
"你这个年纪，35岁，上有老下有小的阶段。
 事业上是想突破但没机会，家里是事儿多但没精力管。"

"你这个盘，16、17年那会儿应该有个节点。
 要么是那几年结的婚，要么是那几年换的工作。"

"子女上看，你这个盘子女宫不算旺。
 要么是还没有，要么是有了但比较操心。"

"结婚有些年头了吧？七八年了？
 这两年夫妻之间是不是有点平淡，不像刚结婚那会儿了。"
```

---

### 40-50岁 (1976-1986年出生)

**Primary concerns**: 中年危机、健康、孩子教育、父母养老

| Age | Key Events | Assertion Examples |
|-----|------------|-------------------|
| 42-45岁 | 中年危机高峰 | "这个阶段，很多人会想这辈子到底要什么" |
| 40-50岁 | 健康问题开始 | "你这个盘，木受克，注意肝胆" |
| 45-50岁 | 孩子青春期/高考 | "孩子应该上初中或者高中了吧" |

**Assertion patterns**:
```
"你这个年纪，孩子应该不小了，上初中还是高中？
 这个阶段，孩子的事儿比自己的事儿还多。"

"健康上看，40岁往后要开始注意了。
 你这个盘，火太旺或者水太弱，心血管和肾脏要注意。"
```

## Reality Assertion Templates

### 婚姻/感情

```
✅ "你这个盘，婚姻宫带X，感情上容易有波折。
    16、17年那会儿应该有个节点，要么成了要么散了。"

✅ "你这个年纪，结婚X年了吧？
    这两年是不是有点平淡，不像刚结婚那会儿了。"

✅ "你这个盘，桃花不旺，感情上应该是那几年定下来的。
    不是那种桃花运不断的人。"
```

### 事业

```
✅ "你这个盘，伤官透出，适合做技术或者脑力活，
    不适合那种按部就班的工作。"

✅ "事业上看，你这几年应该动过换赛道的念头。
    但35岁这个年纪，动不动是个问题。"

✅ "你这个盘，偏财旺，收入应该是时好时坏，
    不稳定但有机会爆发。"
```

### 子女

```
✅ "你这个盘，子女宫一般，要么是还没有，
    要么是有了但比较操心。"

✅ "你这个年纪，孩子应该X岁了吧？
    上学的事儿应该挺费心的。"
```

### 财运

```
✅ "你这个盘，正财不旺偏财旺，
    不适合死工资，适合那种有机会爆发的路子。"

✅ "财运上看，你这个盘是那种前面辛苦后面甜的。
    得到X年之后才见光。"
```

## ⚠️ AVOID These Patterns

### ❌ Psychological Assertions (Use SPARINGLY)

| Avoid | Why |
|-------|-----|
| "心里有点事儿" | Applies to everyone who comes to fortune telling |
| "想动又动不了" | Too vague |
| "憋着一股劲儿" | Barnum effect |
| "精力不够用" | Unverifiable |
| "最近不太顺" | Too generic |

### ❌ Generic Year Assertions

| Avoid | Why |
|-------|-----|
| "22年不太顺" | Not everyone was affected |
| "这两年卡住了" | Too vague |
