# Common Misinterpretations to Avoid

Based on analysis of failed predictions, these are the patterns that lead to inaccurate readings.

---

## ⚠️ Over-Interpretation Patterns

### 1. Five Elements Over-Reading

**Mistake:** Interpreting slight excess as "severe imbalance"

```
❌ "你这个盘水很旺，严重失衡"
   (Water might be just slightly above average)

✅ "你这个盘，水偏旺一些"
   (More accurate, leaves room for nuance)
```

**Rule:** Unless one element is overwhelmingly dominant (>40%), describe it as "偏旺" or "偏弱", not extreme.

### 2. Clash Over-Dramatization

**Mistake:** Interpreting every clash as "major conflict/trauma"

```
❌ "寅申冲，婚姻宫带刑，感情上肯定有问题"
   (Many people with clashes have normal marriages)

✅ "寅申冲，感情上可能有些波动，但你具体怎么样？"
   (Acknowledges the pattern without over-stating)
```

**Rule:** Clash indicates tension/energy, not inevitable disaster. Many successful people have prominent clashes.

### 3. Age Calculation Errors

**Mistake:** Wrong dayun starting age

```
❌ "34岁，正在庚戌大运的尾巴上"
   (Need to verify actual dayun calculation)

✅ "你现在的这步大运..."
   (Without specifying exact age unless verified)
```

**Rule:** Always verify dayun starting age with the tool output, don't calculate mentally.

### 4. Vague Timeframe Generalization

**Mistake:** "这几年" without specific anchor

```
❌ "这几年运气不太好"
   (Applies to everyone, not impressive)

✅ "从22年开始，具体是22、23年那段时间"
   (Specific, verifiable)
```

**Rule:** If you can't pinpoint a year, at least give a specific range with start point.

## 🚫 Patterns That Lead to Wrong Predictitions

### Pattern 1: Assuming Without Verification

**The Trap:**
```
User: "1992年11月22号 早晨五点钟 女"
AI: "你这个盘，婚姻宫带刑，要么晚婚，要么婚姻有问题"

Reality: User is married with children
```

**The Fix:**
- NEVER assume marital/parental status from bazi alone
- Always ask or wait for client to reveal
- Use phrases like "按盘面看..." with qualifier

### Pattern 2: Income Prediction from Single Factor

**The Trap:**
```
AI sees: "时柱丙午偏财坐旺"
AI says: "收入这块儿，应该不算差"

Reality: User says income is poor
```

**The Fix:**
- Wealth star ≠ actual income
- Consider: industry, economy, personal choices
- Don't make definitive income statements without context

### Pattern 3: Cold Reading Without Verification

**The Trap:**
```
AI: "从22年开始，这种感觉应该越来越明显了"
   (Barnum statement - applies to everyone)

User: Doesn't confirm or deny
AI: Continues building on this unverified assumption
```

**The Fix:**
- Cold reading openers need explicit confirmation
- If client doesn't say "对" or nod, assume it didn't land
- Don't stack assumptions on unverified assertions

### Pattern 4: Post-Hoc Rationalization

**The Trap:**
```
User: "我结了婚了，女儿也很大了"
AI: "是我看走眼了...寅申冲，我本来以为应在感情上，现在看应该是应在别的地方"
   (Trying to save face instead of admitting error)
```

**The Fix:**
- Admit error directly: "是我算错了"
- Don't try to retrofit the prediction
- Move on to what you're confident about

## ✅ Correct Interpretation Principles

### Principle 1: Consider the Whole Chart

Don't isolate single factors:

```
❌ "你这个盘水旺，所以..."
✅ "你这个盘，水偏旺，但日主也有根，整体来看..."
```

### Principle 2: Distinguish Fate vs. Luck

```
先天格局 (Fate) = What you're born with
后天运势 (Luck) = How it plays out over time

Don't confuse permanent traits with temporary phases.
```

### Principle 3: Context Matters

Consider:
- Historical era (someone born in 1960 vs 1990)
- Geographic location (urban vs rural)
- Socioeconomic background
- Gender expectations in their culture

### Principle 4: Balanced View

```
命有贵贱，运有好坏
(Fate has levels, luck has ups and downs)

Don't paint everything as either great or terrible.
Most lives have mixed patterns.
```

## 🔍 Self-Check Before Each Assertion

Ask yourself:

1. **Is this based on tool output?** (Not my imagination)
2. **Did client confirm similar statements?** (Not assuming)
3. **Am I over-stating the case?** (Use qualifiers if unsure)
4. **Is there contradictory evidence?** (Acknowledge complexity)
5. **Would this apply to 50%+ of people?** (Too generic if yes)
